# 1. 运行环境
> python 3.6.8  
> solidity 0.4.24  
> ganache  
# 2. 使用
## 合约部署
打开ganache，然后运行deploy_contracts.py代码将合约部署到区块链节点  
```python deploy_contracts.py```

## 运行后端程序
```python app.py```
